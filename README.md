# Veluna Cafe - Frontend

Next.js 14 frontend for Veluna Cafe

## Tech Stack
- Next.js 14
- React
- TypeScript
- Tailwind CSS

## Environment Variables

```env
NEXT_PUBLIC_API_URL=https://your-backend-url.railway.app
```

## Run Locally

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Import GitHub repository
2. Set `NEXT_PUBLIC_API_URL` environment variable
3. Deploy!
